## 3. Recherche de fichiers

### Trouvez des fichiers spécifiques

**À faire :**

- Trouvez tous les fichiers .conf dans `/etc`
- Recherchez "errorll dans `/var/log/syslog` 
- Comptez le nombre de fichiers .txt dans votre home 
- Trouvez les fichiers modifiés aujourd'hui 

### Solution

```bash
find /etc -name "*.conf"
grep "error" /var/log/syslog
find . -type f -name "*.txt" | wc -l
find . -type f -mtime -1
```
