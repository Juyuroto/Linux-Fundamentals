## 1. Navigation et exploration

### Familiarisez-vous avec l'arborescence Linux

**À faire :**

- Affichez votre répertoire courant 
- Allez dans le repertoire /var/log 
- Listez tous les fichiers avec détails 
- Retournez à votre répertoire personnel 
- Affichez l'espace disque disponible

### Solution

```bash
pwd
cd /var/log
ls -la
cd ~
df -h
```
